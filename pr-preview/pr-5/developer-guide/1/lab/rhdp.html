RHDP (skip if using ROLE)

RHDP is our primary lab development platform.

  1. Login to RHDP using your internal SSO credentials.

  2. Identify a suitable lab environment for your course.

  3. Launch and inspect the lab environment.

  4. Note any required changes to fit your needs.

  5. If no lab catalog suits your needs, we can create a new lab environment catalog for you.

For assistance, submit a request via this form (select Open Training).

Refer to the FAQ for more details.
