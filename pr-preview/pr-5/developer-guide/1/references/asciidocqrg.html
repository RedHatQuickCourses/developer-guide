Asciidoc Quick Reference Guide

HEADINGS

Refer to the heading of this page (Asciidoc Quick Reference Guide) for an example of the top level heading. This is used for the navigation (previous/next) of pages in the rendered training content.

Syntax:
= This is a top level heading

This is a second level heading

Syntax:
== This is a second level heading

This is a third level heading

Syntax:
=== This is a third level heading

TEXT

This is example of normal text.
You can write the text in Bold or in italics.
You can use commands.
You can highlight text.

Syntax:
This is example of normal text.
You can write the text in *Bold* or in _italics_.
You can use `commands` (note the backticks).
You can #highlight# text.

For a new paragraph, skip a line.

You may also use hard line breaks like this
for a new line within the paragraph.

Syntax:
For a new paragraph, skip a line.

You may also use hard line breaks like this +
for a new line within the paragraph.

LINKS

Link to an external website with the click here option. (This will open in another tab or window of the browser.)

Syntax:
Link to an external website with https://docs.asciidoctor.org/asciidoc/latest/[click here,window=_blank].

Link to a different page in the same section with the other page within the same section. (This will open in the same browser window.)

Syntax:
Link to another page in the same section with xref:text.adoc[the other page within the same section].

Link to a page in a different section with the other page in the different section. (This will open in the same browser window.)

Syntax:
Link to a page in a different section with xref:demolab:section1.adoc[the other page in the different section].

. Numbered Step 3

  .. Sub Numbered Step 2

Syntax:
* Bullet list item 1
* Bullet list item 2
  ** Sub Bullet list item 1
  ** Sub Bullet list item 2
* Bullet list item 3
